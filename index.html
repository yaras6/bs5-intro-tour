Bootstrap 5 Into Tour free extension

Plain JS extension for Bootstrap 5, which allows to create into/tours easly.

Bootsrap 5 Intro Tour is a extension for bootstrap 5. Moreover this plugin is very lightweight, because it written in plain JS, and don't have jquery dependencies.

If you find any bug or have ideas how can I improve this plugin, please create issue on github.

The extension is free, but any financial support is appreciated.
